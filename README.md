# 🚀 Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Emanshahid22/E-Farming-Website.git
```

## 2️⃣ Move Project to XAMPP

Place the project folder inside:

```bash
htdocs/
```

## 3️⃣ Import Database

- Open phpMyAdmin
- Create a database
- Import SQL file from:

```bash
SQL_Files/
```

## 4️⃣ Start Server

Start:
- Apache
- MySQL

using XAMPP Control Panel.

## 5️⃣ Run Project

Open browser:

```bash
http://localhost/E-Farming-Website
```
