# 🌿 Gift E-Farming: Premium Organic Marketplace
> A full-stack E-Commerce solution connecting local organic farmers with consumers.

Gift E-Farming is a sophisticated web application built with a **glassmorphic UI**. It provides a seamless shopping experience for organic produce and a robust administrative backend for inventory management.

## ✨ Key Features
- **Modern Glassmorphism:** Clean design using backdrop-filter effects and soft shadows.
- **Responsive Grid:** Optimized 13-product grid that scales perfectly across all devices.
- **Admin Dashboard:** Secure portal for system administrators to manage stock and view transactions.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP (MySQLi)
- **Database:** MySQL
- **Design:** FontAwesome 6, Google Fonts (Poppins)

## 📂 Project Structure
- **PHP_Files/**: Core application logic and page structures.
- **CSS_Files/**: Custom stylesheets for premium UI effects.
- **Images/**: High-resolution product and UI assets.
- **SQL_Files/**: Database schema and initial data.
